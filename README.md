# Простая утилита, чтобы быстро проверить, какой DNS-сервер в вашем регионе выдает лучшую задержку. Скрипт умеет подтягивать системные настройки, читать список серверов из csv и делать по несколько замеров для каждого IP через ThreadPool.

# Главная фишка - при каждом запросе генерируется случайный поддомен. Это заставляет DNS-сервер реально искать адрес, а не отдавать закешированное значение, поэтому результаты получаются честными.
